### 1. Removed stray semicolon after <Stack /> in _layout.tsx
- **File:** `forever-client/app/_layout.tsx`
- **Change:** Line 12 — removed trailing `;` after `<Stack screenOptions={{ headerShown: false }} />`
- **Reason:** In JSX inside React Native, a bare `;` character is treated as a text string node. React Native requires all text to be wrapped in a `<Text>` component, so this caused the "Text strings must be rendered within a <Text> component" error.
- **Revert:** Add `;` back after `<Stack screenOptions={{ headerShown: false }} />` on line 12
